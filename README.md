# LightPractice 

It is an app build to practice the different timing of the lights as it is in official matches of the 25-meter events. 

It incudes the following Modes:

|25 Metre Rapid Fire -8s, 6s and 4s series.

|25 Metre Standard Fire -150s, 20s and 10s series

|25 Metre Precision stage 5 minute and Rapid stage 5*3s.
